kmo Ontology Specification

Abstract

Knowledge Mining Ontology (kmo) defines a high-level global schema and provides a vocabulary for users to support automatic knowledge mining through experimental datasets using ontology and ML.

IRI: http://w3id.org/kmo#UnderwaterReboundTest

When the value of the angle is positive, the corrected value is negative, and vice versa; The closer the angle is to the horizontal, the smaller the absolute value of the corrected value; When the test area is a top surface, the corrected value is positive; When the test area is bottom surface, the corrected value is negative; The deeper the water is, the larger the corrected value is; The larger the measured rebound value is, the smaller the absolute value of the corrected value is.
